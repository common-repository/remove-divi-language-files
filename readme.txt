Remove unused language files from the Divi theme

== Description ==

This plugin is an quick and easy way to delete unused language files from your [Divi theme](https://www.elegantthemes.com/gallery/divi/). Deleting these files can save you 15+ MB of diskspace.

== Usage ==

1. Install the plugin
2. Navigate to Settings > Remove Divi Language Files
3. If there are any files to be deleted, check that the correct locales are shown for omission. Otherwise you're already done.
4. Press the "Remove Divi language files" submit button
5. You're done!
